# ReactPen
A code editor app using React and TailwindCSS. In this app, the user can enter code in two different editors. One is for web development technologies which includes HTML, CSS and JavaScript. Other one is for programming languages such as C, C++, Java and Python. User can enter code in any editor and see the output of his code.

This app uses React for the client side code, TailwindCSS and Material-UI for styling, and NodeJS and Express for server side code.

This app was developed as a project given in SMP 2021 by EEESoc. Deployed version can be viewed in the link below

[Demo here](https://next-pen.vercel.app/)

### Commands 

```bash
# Install dependencies (server & client)
npm install
cd client && npm install

# Run server & client (:3000 & :5000)
npm run dev
```
